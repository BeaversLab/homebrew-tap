# BeaversLab Homebrew Tap

Homebrew tap for BeaversLab CLI tools.

## Usage

```bash
brew tap beaverslab/tap
brew install chatlog
brew install x-skill
```

`x-skill` 当前固定到公开的 `v0.1.1` 源码归档，Homebrew 会自动拉取 Rust 作为构建依赖并在本地编译安装。

## Available Formulas

| Formula | Description |
|---------|-------------|
| chatlog | Extract and save agent chat logs (Claude, Codex, Gemini CLI) as local Markdown files |
| x-skill | Install, manage, and discover skill packages for 42+ AI coding agents |
